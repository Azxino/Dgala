PROYECTO: Ecommerce de Dotaciones (Laravel + MySQL + API REST)

CONTEXTO DEL PROYECTO:
Estamos construyendo un ecommerce profesional de dotaciones/uniformes con manejo de inventario avanzado por lonas.

OBJETIVO:
Crear una API REST profesional que permita:
- productos
- variantes (talla/color)
- imágenes
- categorías
- inventario por lonas
- carrito
- órdenes
- pagos
- envíos
- devoluciones
- usuarios con roles
- cupones
- notificaciones

STACK / HERRAMIENTAS:
Backend:
- Laravel
- PHP
- MySQL = http://localhost:8080 / (root . root)
- phpMyAdmin
- Frontend: Vue.js

php artisan tinker

salir:
exit
