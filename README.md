# D'gala: E-commerce de Uniformes Mayoristas
Autor: Azx

# Descripción
D-gala es un proyecto de página web de e-commerce orientada a la venta mayorista de uniformes.
El sitio permite a los clientes explorar productos, conocer sus características y realizar pedidos al por mayor de manera sencilla.

# Funcionalidades
- Catálogo de productos con imágenes y descripciones.
- Información de tallas y disponibilidad de stock.
- Carrito de compras para pedidos mayoristas.
- Sección de contacto para consultas y cotizaciones.

# Tecnologías utilizadas
- HTML y CSS  para la estructura y diseño del sitio.
- JavaScript para funcionalidades interactivas.
- PHP para Backend y logica
- MYSQL Base de datos
- Git y GitHub para control de versiones y gestión del proyecto.
- Laravel /Backend
- Vue.js  /Frontend
- Docker  /Contenedor
  
  

## Objetivo
El objetivo del proyecto es ofrecer una plataforma intuitiva y funcional para la venta mayorista de uniformes, facilitando la gestión de pedidos y la comunicación con los clientes.
